## Usage

To download the [Ky Alucard colour mod](https://gamebanana.com/mods/513718) and register it being used over Ky Kiske.

```sh
ggmod --char KYK https://gamebanana.com/mods/513718
```

## Installation

```sh
cargo install ggmod
```
